# Aurum-Elegance---Timeless-Luxury-Fashion
Aurum Elegance is a sophisticated, responsive HTML/CSS landing page designed for a luxury fashion brand. It showcases high-end watches, jewelry, home decor, and couture items with an elegant aesthetic that reflects timeless refinement.

🔧 Installation & Usage
Option 1: Direct Use
Download both index.html and style.css files

Place them in the same directory

Open index.html in your browser

Option 2: Single File
Copy the complete HTML from the provided code

Save as index.html

Open in browser (all styles are embedded)

Option 3: Customization
Replace placeholder images with actual product photos

Update contact information and social links

Modify product items in the grid

Adjust color variables in :root for brand customization

Add functionality to login form (currently static)

💡 Customization Tips
Colors
Modify the :root variables in CSS:

css
:root {
  --gold: #d4af37;        /* Primary gold */
  --gold-dark: #aa7c2c;   /* Darker gold */
  --dark-bg: #1c1815;     /* Dark background */
  --cream: #fef8f0;       /* Cream background */
}
Products
Update the product grid with your own items:

html
<div class="product-card">
  <div class="product-img"><i class="fas fa-YOUR-ICON"></i></div>
  <div class="product-info">
    <div class="product-title">Product Name</div>
    <div class="product-category">Category</div>
    <div class="product-price">$Price</div>
  </div>
</div>
